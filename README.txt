Currency Converter

Автор: Афанасьев Александр Александрович

Описание
GUI-приложение "Currency Converter" для конвертации валют с использованием внешнего API (ExchangeRate-API),
сохранением истории конвертаций в JSON и возможностью её загрузки при старте программы.
Приложение написано на Python с использованием библиотеки tkinter и модуля requests.

Как получить API-ключ
1. Перейдите на сайт https://www.exchangerate-api.com/ .
2. Нажмите "Sign Up" и зарегистрируйтесь (можно использовать бесплатный тариф Free Plan).
3. После регистрации в личном кабинете вам будет выдан персональный API-ключ вида xxxxxxxxxxxxxxxxxxxx.
4. Скопируйте этот ключ и введите его в приложении, нажав кнопку "Указать API-ключ".

Примечание: Бесплатный тариф имеет ограничения по количеству запросов и обновлению курсов.
Подробности смотрите в документации ExchangeRate-API (раздел Docs на сайте).

Возможности
- Выбор исходной и целевой валют (USD, EUR, RUB, GBP, JPY, CNY, CHF, CAD, AUD).
- Ввод суммы для конвертации.
- Конвертация по актуальному курсу с помощью ExchangeRate-API.
- Проверка корректности ввода: сумма должна быть положительным числом.
- Отображение результата конвертации в отдельном поле.
- Таблица истории конвертаций (дата/время, из какой валюты, в какую, сумма, результат, курс).
- Сохранение истории в файл history.json и загрузка при старте программы.

Требования
- Python версии 3.10 или выше.
- Доступ в Интернет для запросов к ExchangeRate-API.
- Установленный модуль requests:
  pip install requests

Запуск
1. Откройте терминал (командную строку) в папке проекта.
2. При необходимости установите зависимости:
   pip install requests
3. Запустите приложение командой:
   python main.py
4. В интерфейсе нажмите кнопку "Указать API-ключ" и введите свой ключ ExchangeRate-API.
5. Выберите валюты "Из" и "В", введите сумму и нажмите "Конвертировать".

Примеры использования
- Конвертация 100 USD в EUR.
- Конвертация 1000 RUB в USD.
- Конвертация 50 EUR в GBP.
После каждой конвертации запись добавляется в таблицу истории и сохраняется в history.json.

Хранение истории
- history.json — файл, содержащий список всех совершённых конвертаций с полями:
  дата и время конвертации, исходная и целевая валюта, исходная сумма,
  результат конвертации, использованный курс.
При следующем запуске программы история загружается и отображается в таблице.
